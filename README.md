# SPFX HelloWorld WebPart

A simple SharePoint Framework (SPFX) HelloWorld web part demonstrating basic SPFX development.

## Prerequisites

- Node.js 16.x
- npm 7.x
- Gulp CLI

## Installation

```bash
npm install
```

## Development

To start the development server:

```bash
npm run serve
```

The workbench will be available at `https://localhost:4321/temp/workbench.html`

## Build

To build the project:

```bash
npm run build
```

## Package

To create a package for deployment:

```bash
npm run bundle
npm run package-solution
```

## Features

- Simple React-based webpart
- TypeScript support
- SPFX 1.17.0
